Docker-MongoDB
</h2>

### Installation

1. Clone the repo
   ```sh
   git clone https://github.com/hwJames/Docker-MongoDB.git
   git pull
   ```

2. Edit Setting
   ```sh
   vi .env
   vi docker-compose.yml
   vi mongo-init.js
   ```

3. Docker Compose Start
   ```sh
   docker-compose up
   ```
